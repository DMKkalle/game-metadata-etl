 # Exjobb – Worko (metadata/arkiv)

## Syfte
Proof of Concept för tillförlitlig metadata: insamling, normalisering och AI-stödd berikning.

## Struktur
- docs/ – anteckningar, mallar, beslut
- data/raw/ – **IGNORERAS** (ingen rådata i Git)
